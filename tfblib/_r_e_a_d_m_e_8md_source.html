1 # Tfblib (Tiny Framebuffer Library)
2 
3 [![Build Status](https://travis-ci.org/vvaltchev/tfblib.svg?branch=master)](https://travis-ci.org/vvaltchev/tfblib)
4 
5 `Tfblib` is a simple and low-level graphics library for drawing to the Linux
6 framebuffer. Currently, it is capable of drawing lines, rectangles and text.
7 It has support both for embedded (compiled-in) fonts in the library and
8 dynamically loaded `PSF` fonts at runtime. In addition to drawing functions,
9 `Tfblib` has a minimal support for keyboard input that allows simple
10 applications to put the TTY input in *raw mode* and read keystrokes. Both
11 blocking and non-blocking modes are supported.

13 Building
14 ---------
15 
16 Building `Tfblib` as a static library is simple as executing (in project's root
17 directory):
18 
19 ```
20 $ mkdir build
21 $ cd build
22 $ cmake ..
23 $ make
24 ```
25 The `make` command will build the library along with the programs in the
26 examples/ directory. In case a release build (with optimizations) is desired,
27 the `cmake` command has to be run this way (assuming the current working
28 directory is the build directory):
29 
30  cmake -DCMAKE_BUILD_TYPE=Release ..
31 
32 Or:
33 
34  cmake -DCMAKE_BUILD_TYPE=RelWithDebInfo ..
35 
36 In case a release build with debug info is desired.

38 A "hello world" application
39 -----------------------------
40 
41 Using `Tfblib` is extremely simple. Here below, there's an example about how
42 to initialize the library and draw a rectangle at the center of the screen.
43 
44 ```C
45 int main(int argc, char **argv)
46 {
47  int rc;
48 
49  if ((rc = tfb_acquire_fb(0, NULL, NULL)) != TFB_SUCCESS) {
50  fprintf(stderr, "tfb_acquire_fb() failed with error code: %d\n", rc);
51  return 1;
52  }
53 
54  uint32_t w = tfb_screen_width();
55  uint32_t h = tfb_screen_height();
56  uint32_t rect_w = 200;
57  uint32_t rect_h = 200;
58 
59  /* Paint the whole screen in black */
60  tfb_clear_screen(tfb_black);
61 
62  /* Draw a red rectangle at the center of the screen */
63 
64  tfb_draw_rect(w / 2 - rect_w / 2, /* x coordinate */
65  h / 2 - rect_h / 2, /* y coordinate */
66  rect_w, /* width */
67  rect_h, /* height */
68  tfb_red /* color */);
69 
70  getchar();
71  tfb_release_fb();
72  return 0;
73 }
74 ```
