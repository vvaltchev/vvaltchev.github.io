1 # Tfblib (Tiny Framebuffer Library)
2 
3 [![Build Status](https://travis-ci.org/vvaltchev/tfblib.svg?branch=master)](https://travis-ci.org/vvaltchev/tfblib)
4 
5 `Tfblib` is a simple and low-level graphics library for drawing to the Linux
6 framebuffer. Currently, it is capable of drawing lines, rectangles and text.
7 It has support both for embedded (compiled-in) fonts in the library and
8 dynamically loaded `PSF` fonts at runtime. In addition to drawing functions,
9 `Tfblib` has a minimal support for keyboard input that allows simple
10 applications to put the TTY input in *raw mode* and read keystrokes. Both
11 blocking and non-blocking modes are supported.

13 Building
14 ---------
15 
16 Building `Tfblib` as a static library is simple as executing (in project's root
17 directory):
18 
19  - mkdir build
20  - cd build
21  - cmake ..
22  - make
23 
24 The `make` command will build the library along with the programs in the
25 examples/ directory. In case a release build (with optimizations) is desired,
26 the `cmake` command has to be run this way (assuming the current working
27 directory is the build directory):
28 
29  cmake -DCMAKE_BUILD_TYPE=Release ..
30 
31 Or:
32 
33  cmake -DCMAKE_BUILD_TYPE=RelWithDebInfo ..
34 
35 In case a release build with debug info is desired.
